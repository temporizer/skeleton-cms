skeleton cms

Skeleton CMS is a flat file cms meaning that there is not database. Skeleton uses PHP write file ability to create content and functionality. In most cms systems your content is submitted to a database and then when needed is called upon for display, instead skeleton writes a series of HTML files that are created on the fly in the admin section. These HTML files can either be updated by overriding them from the admin section or going directly to them on your server.

Content is created via modules that are loaded into grid positions laid out by Skeleton CSS Framework. Icons from Font Awesome are preloaded as a plugin for use. You can learn more about either by clicking the links provided.
